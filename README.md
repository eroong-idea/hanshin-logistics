# 한신 주문내역 조회

월별 · 가맹점별 **전용/범용 품목 발주 현황**을 조회하는 웹앱입니다.

## 팀원 사용법
아래 주소를 열기만 하면 됩니다 (설치·업로드 불필요):

> https://<아이디>.github.io/<저장소이름>/

- 상단 **가맹점 검색**으로 가맹점 선택 → 월별 전용/범용 수량·주문액과 전용품목 상세 확인
- **기간** 선택으로 특정 월 / 전체 기간 조회
- **새로고침** 버튼으로 최신 데이터 반영

## 관리자: 매월 데이터 올리는 법 (GitHub Desktop)
1. 이 폴더의 **`data/`** 안에 새 달 엑셀을 넣습니다. (예: `주문내역_7월.xlsx`)
   - 파일명이나 시트명에 **월(6월, 7월 …)** 이 들어가야 자동으로 구분됩니다.
2. **GitHub Desktop**을 열면 바뀐 파일이 보입니다 → 아래 **Summary**에 메모 입력 → **Commit** → 우측 상단 **Push origin**.
3. 잠시 후 팀원 화면에서 **새로고침** 하면 반영됩니다.

## 파일 구성
- `index.html` — 웹앱 본체 (GitHub Pages가 이 파일을 서비스)
- `data/` — 월별 주문 엑셀 폴더 (여기에 파일을 넣으면 앱이 자동으로 읽음)
- `한신_주문조회.html` — 내 컴퓨터에서 바로 열어 쓰는 로컬용 (동일 기능)
